Aim: To determine the focal length of a convex lens.

Apparatus:
Lenses of various focal lengths

Optical bench

Screen

Light box

Procedure:
Fix the convex lens in the holder and place it securely on the optical bench.

Switch on the light box.

Move the screen (and the lens, if required) until you get a sharp image of
the illuminated cut-out in the light box.

Record the different distances, namely the object distance and the screen distance.

Calculate the focal length of the convex lens from the given equation.

Repeat the experiment with a lens of different focal length.

Note: The following calculations are applicable only when you see a sharp inverted image of the object on the moveable screen.

Object distance from the lens, So =173    units

Object height, Ho =30     units

Screen distance from the lens when the image is sharp, Si =117.57     units

Focal length, f, is given by the equation:

1/f = 1/So + 1/Si

Therefore,

Focal length, f = ( So x Si ) / ( So + Si ) =
(173     x 117.57   ) / ( 173    + 117.57   )

Focal length, f = 70   units

Image Height, Hi:

So/Ho = Si/Hi

Therefore, Hi = (Ho/So) x Si = (30  / 173    )  x 117.57

Image height, Hi = 20.39     units
